I Need to Escapee

Team of 5 | Game Designer | 4 months

Escapee is a horror-comedy escape game where players need to urgently use the washroom and escape

Core Challenges

Motivation

How to motivate the players to finish the game?

Make Mechanics

How to make mechanics out of needing to go to the bathroom?

Modern 2D RPG

How to create an old-school game while keeping a modern style?

Core Solutions

5-minute Timer

The player's bladder has a 5-minute timer. Short gameplay while keeping players on their toes

Puzzles

All bathroom items are scattered across rooms. You have to find them before relieving the timer

Consistent Graphics

Minimalistic yet colorful pixel art that mimics the referenced environment.

Laboratory

The puzzle is simple to obtain soap:
1. Get water from the sink using the empty beaker
2. Obtain the oil and lye on the table
3. Interact with the big beaker to combine all ingredients.

On a technical level, this section needed to check whether players has done the following steps. In this case, variables were used to keep track of state.

Storage Room

Upon interacting with the boxes, it leads the player to what they have to do: kick the boxes down to get the plunger.

This section only needed to keep track of the number of space presses in the box collider (where players interact with the box).

After a certain number, change the background and include an obtainable plunger.
