Luminth

Team of 3 | Software Developer C# | 72 hours

Summon light to find your way out of the dark, dangerous maze. Avoid becoming a fish’s snack! Submitted for Ludum Dare 55.

Core Challenges

Working with another dev

How to communicate our ideas and implementation without stepping on each other's toes?

Unique core mechanics

With a unique idea that doesn't have any direct references, how do we build it AND teach players?

Aesthetic Polish

How to make holes on an image to reveal the background?

Core Solutions

Backlog and Branches

Using concepts from Agile Collaboration, Figma for backlogging tasks, and directly translating them into a Github Branch.

Playtests and Tutorials

Fixing bugs through playtesting as a clueless player, while creating better UX through a tutorial level for players to feel comfortable.

Stencil Buffer

Taking advantage of Unity Textures and stencil buffer to make inverted holes into a texture.

Backlog and Branches

Given the short time, it was idiotic to set up a complex and traditional standard of communication for a quick temporary project.

By using Figma whiteboard, we jotted all our rough ideas, and quickly identified small tasks we could start with to make quick progress.

By working in branches that represent such small tasks, we minimized merge conflicts and stayed goal-oriented, while jotting down more tasks that needed to be implemented along the way.

Playtests and Tutorials

Since we've never made a blind-maze, we made it a priority to playtest our game at the early stages in order to ensure the game was not too hard yet not too easy.

To ensure that players could understand the mechanics however, it was easiest through a tutorial.

1) Identify each mechanic to be taught
2) Rearrange them to be introduced in a linear manner
3) Format it as a story that makes the mechanics naturals in the game's context
4) Create a dedicated scene for the tutorial
5) Create a Manager script that listens for events to trigger the introduction of a mechanic
6) Create Game Events and collisions that raise events for the Manager to hear

Stencil Buffer

By taking priority in stencil operation, yet taking alpha clip and 0 color mask, it can make part of the texture transparent.

This made a "Hole Object" that could be placed anywhere in the scene like a regular object, making it easy to customise and automatically place.
